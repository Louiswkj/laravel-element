## 博客地址

[Laravel5.5 + Vue2 + Element 环境搭建]

## 安装步骤

1.  git clone git@github.com:mrzhouxiaofei/Larvuent.git
2.  composer install
3.  cp .env.example .env
4.  php artisan key:generate
5.  npm install
6.  npm run dev
